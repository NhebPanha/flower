## 🚀 About Project

* 🧱 Clean Architecture
* 🔁 Bloc State Management
* ☁️ Supabase Database

---

## ⚙️ Tech Stack

* Flutter SDK: `3.35.7`
* Dart SDK: `>=2.19.0 <3.0.0`
* Architecture: Clean Architecture
* State Management: Bloc
* Backend: Supabase
